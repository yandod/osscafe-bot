# これは何

Line beaconに接触したユーザーの情報をデータベースに記録するサンプルです。

# 必要な物

- Line beacon
- Line@ アカウント
- 上記にリンクした Channel (BOT)

# 注意

- 小規模の利用を想定
- ビーコンのイベントは頻繁に発生するので最後の記録から1時間以内の場合は何もしない
